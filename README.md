# Ridefinder
## To use
```bash
$npm install
$bower install
$gulp
```
## Notes
* Running ```$gulp ``` will minify and concatenate files and start the node server with nodemon
** If you don't have nodemon, run ```$npm install -g nodemon```
* Other gulp tasks can be found in the gulpfile.js
